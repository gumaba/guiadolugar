How to change the logo?

You can do that very easily, simply replace the current logo image with your own one, the curren logo is located in /img/logo.png folder. The required size is 222 × 43 (px)

How to change menu?

The menu code is located on the 45 line, index.html file.
You can easily change the menu names, add or delete them from the code.

How to manage home slider?

The home gallery start from line: 144 (index.html file). Let's take a look what we have inside:

<div class="oneByOne_item ls_bg5">
<img src="img/slider/slider1/ls5_txt1.png" class="ls5_txt1" data-animate="fadeInUp">
<img src="img/slider/slider1/ls5_txt2.png" class="ls5_txt2" data-animate="fadeInUp">
<img src="img/slider/slider1/ls5_txt3.png" class="ls5_txt3" data-animate="fadeInUp">
</div>

img/slider/slider1/ls5_txt1.png - is the path to the slider images. You can add your own images to the slider by replacing the existing one or add your own ones. To better understand on how "OnebyOne" slider works, please read the documentation located in the "License" folder. Right there you will find the information on how to add the images to the slider and change its settings.

How to change the color of the theme?

You can do that easily,there are 3 ways:

  • Auto replacing - please open css/color-default.css and change the main color ffd600 to your own one
  • Manually - please open css/colors.css and replace the main color ffd600 to your own one in the following lines: 3, 4, 27, 37, 70, 79, 106, 112
  • Use ready-made CSS files that are located in the\css\colors folder. Simply rename one of 9 premade css files into the color-default.css and use it.

Contact Form Setup

You can setup your contact form in 2 clicks, please open mail.php file and find "test@test.com", simply replace it with your email address and all the requests will be sent to it. That's all ;)

Source files included
